Le Dark Web

Le nom

Le dark web (signifiant "web sombre") fait partie de la partie cachée d'internet. Les pages web disponibles sur celui-ci ne sont pas directement accessible depuis un navigateur web lambda.

Accès au dark web

Comme dit précédemment, le dark web n'est pas accessible depuis un navigateur web normal. Par exemple, il n'est pas possible d'aller sur le dark web avec Google Chrome ou Firefox.
